EMSE 4571: Intro to Programming for Analytics




This is the course website for “EMSE 4571: Intro to Programming for Analytics” at the George Washington University. This is the first part of a two-course sequence designed to provide a foundation in programming for data analytics using the R programming language:

  • Course 1: EMSE 4571: Intro to Programming for Analytics
  • Course 2: EMSE 4572: Exploratory Data Analysis
