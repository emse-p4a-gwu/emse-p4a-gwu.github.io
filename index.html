Programming for Analytics


Description:

This is the course website for the EMSE undergraduate course “Programming for Analytics” at the George Washington University.

This is the first part of a two-course sequence designed to provide a foundation in programming for data analytics using the R programming language:

  • Course 1 (Fall Semester): Programming for Analytics
  • Course 2 (Spring Semester): Exploratory Data Analysis
